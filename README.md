Front-end Boilerplate
=========================

Technologies used:
-
- HTML5 + CSS3
- JavaScript with CoffeeScript

Third party tools and libraries used:
-
- jQuery
- Bootstrap Theme: "Scale" by flatfull
- Grunt
- Browserify (using: caching-coffeeify and hbsfy)
- Backbone
- Marionette

Steps
-
1. npm install
2. npm start
3. open http://localhost:8315
